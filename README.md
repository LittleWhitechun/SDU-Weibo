# SDU-Weibo

Jquery做的类似微博的小项目 

**UI框架：**Bootstrap

**数据存储：**MYSql 用户数据、微博数据等需要连接数据库

**后端:**php

#### 实现功能：

- 登录/注册
- 发帖/查看帖子
- 查看图片
- 大图浏览
- 点赞评论回复转发
- 个人资料/个人等级
- 热搜榜
